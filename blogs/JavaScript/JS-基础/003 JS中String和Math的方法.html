# String

# 1.charAt()：从一个字符串中返回指定位置的字符

const str = 'Hello World!';
console.log(str.charAt(6)); // W

const str = 'Hello World!';

# 3.indexOf()：返回调用它的 String 对象中第一次出现的指定值的索引

从 fromIndex 处进行搜索。如果未找到该值，则返回 -1

如果查找的字符串为空，且 fromIndex 值小于字符串长度，则返回 fromIndex 的值

const str = 'Hello World!';
console.log(str.indexOf(''));    // 0
console.log(str.indexOf('', 0)); // 0
console.log(str.indexOf('', 3)); // 3

const str = 'Hello World!';
console.log(str.indexOf('', 12)); // 12
console.log(str.indexOf('', 13)); // 12

# 4.slice()：提取某个字符串的一部分，并返回一个新的字符串，且不会改动原字符串

const str = 'Hello World!';
console.log(str.slice(2)); // llo World!
console.log(str.slice(2, 5)); // llo
console.log(str.slice(-6)); // World!

# 5.substr 和 substring 的区别

substr(start[,length])：返回一个字符串中从指定位置开始到指定字符数的字符

substring(indexStart[,indexEnd])：返回一个字符串在开始索引到结束索引之间的一个子集, 或从开始索引直到字符串的末尾的一个子集（推荐使用该方法）

相同点

当只有一个参数时，两者的功能是一样的，返回一个从指定位置开始到字符串结束的子字符串

不同点

  • 如果 start >0，且大于或等于字符串的长度，则 substr 返回一个空字符串
  • 如果 start <0，则从字符串末尾开始计算。如果 start 为负值且 abs(start) 大于字符串的长度，则 substr 使用 0 作为开始提取的索引
  • 如果 substr =0，则返回整个字符串
  • 如果 length 为 0 或负值，则 substr 返回一个空字符串。如果忽略 length，则 substr 提取字符，直到字符串末尾
  • 如果 indexStart 等于 indexEnd，substring 返回一个空字符串
  • 如果任一参数小于 0 或为 NaN，则被当作 0
  • 如果任一参数大于字符串长度，则被当作字符串长度
  • 如果 indexStart 大于 indexEnd，则将两个参数交换位置

# 6.split()：返回指定字符分隔的字符串数组

const str = 'Hello World!';
console.log(str.split(' ')); // ["Hello", "World!"]
console.log(str.split(' ', 1)); // ["Hello"]

使用 split 反转字符串

const str = 'Hello World!';
console.log(str.split('').reverse().join(''));

# 7.replace()：返回一个由替换值（replacement）替换一些或所有匹配的模式（pattern）后的新字符串

模式可以是一个字符串或者一个正则表达式，替换值可以是一个字符串或者一个每次匹配都要调用的回调函数，不会改变原字符串

const str = 'Hello World!';
console.log(str.replace(/\s/, '')); // HelloWorld!
1
2

在 replace() 中使用 global 和 ignore 选项

const str = 'Hello World | Hello world';
let re = /world/gi;
console.log(str.replace(re, 'JavaScript')); // Hello JavaScript | Hello JavaScript

2
3

g 表示全局替换，i 表示忽略大小写
