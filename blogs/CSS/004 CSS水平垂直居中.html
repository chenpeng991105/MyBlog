# 1、利用定位和 transform 属性实现

<style>
    .parent{
        width: 500px;
        height: 500px;
        background-color: cornflowerblue;
        position: relative;
    }
    .child{
        position: absolute;
        top: 50%;
        left: 50%;
        transform: translate(-50%, -50%);
    }
</style>

        width: 500px;
        height: 500px;
        background-color: cornflowerblue;
        display: flex;

        align-items: center;
    }

        background-color: red;
