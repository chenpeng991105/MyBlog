# 1.props/$emit

单向数据流，在父组件中定义传递给子组件的数据，在子组件中使用 props 接收

在子组件中通过 $emit 方法给父组件传递消息，修改父组件中的数据

<!--父组件-->
<template>
  <div>
    <input type="text" v-model="name">
    <Child :name="name" @nameChanged="updateName"/>
  </div>
</template>

<script>
import Child from "./components/Child";
export default {
  components: {
    Child
  },
  data(){
    return{
      name: 'cyy'
    }
  },
  methods: {
    updateName(val){
      this.name = val
    }
  }
}
</script>

# 3.$attrs/$listeners

跨级组件传递数据，如果不需要处理，可以使用这种方法

<!--父组件-->
<template>
  <div class="parent">
    <child-a :name="name" :age="age"/>
  </div>
</template>

<script>
import ChildA from "./components/ChildA";
export default {
  components: {
    ChildA,
  },
  data(){
    return{
      name: 'cyy',
      age: 22
    }

其中 inheritAttrs 属性：

前提：子组件的 props 中未注册父组件传递过来的属性

  1. 当设置 inheritAttrs: true (默认)时，子组件的顶层标签元素会渲染出父组件传递的属性
  2. 当设置 inheritAttrs: false 时，子组件的顶层标签元素不会渲染出父组件传递的属性
  3. 不管设置为 true 还是 false，子组件中都能通过 $attrs 属性获取到父组件中传递的属性

# 5.$parent/$children、ref

<template>
  <div class="parent">
    <child-a ref="childA"/>
  </div>
</template>

<script>
import ChildA from "./components/ChildA";
export default {
  components: {
    ChildA,
  },
  mounted() {
    const childA = this.$refs.childA
    console.log(childA.name)// vue
    childA.say() // Hello Vue
  }
}
</script>
